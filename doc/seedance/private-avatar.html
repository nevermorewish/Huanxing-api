虚拟人像素材

私域虚拟人像素材提交接口，用于批量提交人像素材进行审核。审核通过的素材会得到一个 asset://... 形式的 Asset URL，可直接作为参考图片传入 Seedance 2.0 视频生成接口，无需重新上传和审核。

  • 支持批量提交，单次最多 20 个素材。
  • 自动创建或复用素材组，返回任务 ID 用于状态查询。
  • 审核通过的素材可直接用于 Seedance 2.0 视频生成。

接口地址

POST /v1/seedance2/private-avatar

package main

import (
	"bytes"
	"net/http"
)

func main() {
	body := []byte(`{
  "group": { "name": "virtual-avatar-group", "description": "demo group" },
  "project_name": "default",
  "asset_type": "Image",
  "assets": [
    { "url": "https://example.com/avatar-a.png", "name": "avatar-a" },
    { "url": "https://example.com/avatar-b.png", "name": "avatar-b" }
  ]
}`)
	req, _ := http.NewRequest("POST",
		"https://api.huanxing.ai/v1/seedance2/private-avatar",
		bytes.NewBuffer(body))
	req.Header.Set("Authorization", "Bearer sk-你的令牌")
	req.Header.Set("Content-Type", "application/json")
	resp, _ := http.DefaultClient.Do(req)
	defer resp.Body.Close()
}
请求将直接发送到 https://api.huanxing.ai，令牌仅用于本次浏览器请求，不会上传或保存。

成功响应返回审核任务信息：

认证

所有接口均需使用 Bearer Token 认证。在请求头中加入：

Authorization: Bearer sk-你的令牌

请求参数

参数类型说明
groupobject素材组信息。不传 group_id 时，服务端会根据此字段自动创建 AIGC 类型素材组。含 name（组名称）与 description（组描述）。与 group_id 二选一，不可同时传入。
group_idstring已有素材组 ID。传入时跳过素材组创建，直接向该素材组提交素材。与 group 二选一，不可同时传入。
project_namestring项目名称，默认 default。
asset_typestring素材类型，默认 Image。可选 Image（图片）/ Video（视频）/ Audio（音频）。
assetsarray素材列表，支持一次提交多个素材，单次最多 20 个。元素含 url（公网可访问素材 URL，必填）与 name（素材名称，必填）。
urlstring单素材兼容写法：素材 URL。与 assets 数组二选一，适用于只提交一个素材的场景。
namestring单素材兼容写法：素材名称。与 assets 数组二选一。

响应

字段类型说明
codeinteger响应状态码，成功时为 200。
data.idstring本地任务 ID，用于后续查询素材审核状态。
data.objectstring任务对象类型，固定为 seedance.avatar.asset.task。
data.statusstring任务初始状态，提交后为 processing。
data.progressinteger任务进度（0 ~ 100）。
data.modelstring当前使用的模型名称。

使用场景

场景 1：批量提交素材（自动创建素材组）

不传 group_id，服务端自动创建 AIGC 类型素材组后提交素材。

{
  "group": {
    "name": "virtual-avatar-group",
    "description": "demo group"
  },
  "project_name": "default",
  "asset_type": "Image",
  "assets": [
    { "url": "https://example.com/avatar-a.png", "name": "avatar-a" },
    { "url": "https://example.com/avatar-b.png", "name": "avatar-b" }
  ]
}

场景 2：向已有素材组追加素材

传入 group_id，跳过素材组创建直接提交。

{
  "group_id": "group_xxx",
  "project_name": "default",
  "asset_type": "Image",
  "assets": [
    { "url": "https://example.com/avatar-a.png", "name": "avatar-a" }
  ]
}

场景 3：单素材兼容写法

只提交一个素材时，可直接使用顶层 url 和 name 字段。

{
  "group_id": "group_xxx",
  "url": "https://example.com/avatar.png",
  "asset_type": "Image",
  "name": "avatar-1"
}

查询审核结果

素材提交后为异步审核任务，使用获取任务状态接口查询：

GET /v1/tasks/{id}

全部通过

{
  "code": 200,
  "data": {
    "id": "task_01K...",
    "status": "completed",
    "progress": 100,
    "result": {
      "assets": [
        { "asset_id": "asset_a", "asset_url": "asset://asset_a", "status": "Active" },
        { "asset_id": "asset_b", "asset_url": "asset://asset_b", "status": "Active" }
      ],
      "usable_assets": [
        { "asset_id": "asset_a", "asset_url": "asset://asset_a", "status": "Active" },
        { "asset_id": "asset_b", "asset_url": "asset://asset_b", "status": "Active" }
      ],
      "failed_assets": []
    }
  }
}

部分失败

批量提交时，只要有一个素材审核失败，任务状态为 failed。已通过的素材仍可用，出现在 result.usable_assets 中。

{
  "code": 200,
  "data": {
    "id": "task_01K...",
    "status": "failed",
    "progress": 100,
    "result": {
      "assets": [
        { "asset_id": "asset_a", "asset_url": "asset://asset_a", "status": "Active" },
        { "asset_id": "asset_b", "asset_url": "asset://asset_b", "status": "Failed" }
      ],
      "usable_assets": [
        { "asset_id": "asset_a", "asset_url": "asset://asset_a", "status": "Active" }
      ],
      "failed_assets": [
        { "asset_id": "asset_b", "asset_url": "asset://asset_b", "status": "Failed" }
      ]
    },
    "error": {
      "code": "task_failed",
      "message": "部分素材审核失败"
    }
  }
}
  • result.usable_assets[].asset_url 可直接用于 Seedance 2.0 视频生成。
  • result.failed_assets 中的素材需更换源文件或重新提交。
  • 单素材任务也会兼容返回 result.asset_url。

审核后使用素材

审核通过后，将 asset://... URL 直接传入 Seedance 2.0 视频生成接口：

{
  "model": "doubao-seedance-2.0",
  "prompt": "让人物在城市街道自然行走",
  "image_urls": ["asset://asset_a"],
  "duration": 5,
  "resolution": "720p"
}

服务端识别到 asset:// 前缀后，会直接提交官方生成任务，不会再次触发素材审核。
