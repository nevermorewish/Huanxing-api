真人人像素材

真人人像素材接口通过同一入口完成三种操作：创建真人认证 H5 会话、查询认证结果、批量提交真人素材进行审核。审核通过的素材会得到 asset://... 形式的 Asset URL，建议配合 doubao-seedance-2.0-face 或 doubao-seedance-2.0-fast-face 模型使用。

  • 步骤一：创建真人认证 H5 会话，获取 h5_link 和 byted_token。
  • 步骤二：用户完成 H5 认证后，查询认证结果获取素材组 group_id。
  • 步骤三：向 group_id 批量提交真人素材进行审核。

接口地址

POST /v1/seedance2/real-avatar

认证

所有接口均需使用 Bearer Token 认证。在请求头中加入：

Authorization: Bearer sk-你的令牌

步骤一：创建真人认证会话

请求参数

参数类型说明
callback_urlstring，必填认证完成后的回调 URL。用户完成 H5 认证后，系统会向该地址发送回调通知。
project_namestring项目名称，默认 default。

调用示例

POST /v1/seedance2/real-avatar 
curl https://api.huanxing.ai/v1/seedance2/real-avatar \
  -H "Authorization: Bearer sk-你的令牌" \
  -H "Content-Type: application/json" \
  -d '{
    "callback_url": "https://example.com/callback",
    "project_name": "default"
  }'

package main

import (
	"bytes"
	"net/http"
)

func main() {
	body := []byte(`{
  "callback_url": "https://example.com/callback",
  "project_name": "default"
}`)
	req, _ := http.NewRequest("POST",
		"https://api.huanxing.ai/v1/seedance2/real-avatar",
		bytes.NewBuffer(body))
	req.Header.Set("Authorization", "Bearer sk-你的令牌")
	req.Header.Set("Content-Type", "application/json")
	resp, _ := http.DefaultClient.Do(req)
	defer resp.Body.Close()
}
请求将直接发送到 https://api.huanxing.ai，令牌仅用于本次浏览器请求，不会上传或保存。

成功响应返回会话任务 ID：

{
  "code": 200,
  "data": {
    "id": "task_01K...",
    "status": "completed"
  }
}

查询该任务可获得 H5 认证链接与认证令牌：

GET /v1/tasks/task_01K...
{
  "code": 200,
  "data": {
    "id": "task_01K...",
    "status": "completed",
    "result": {
      "byted_token": "token_xxx",
      "h5_link": "https://...",
      "raw_response": {}
    }
  }
}

步骤二：查询真人认证结果

用户完成 H5 认证后，通过 GET /v1/tasks/{id} 查询步骤一任务获取 byted_token，再用此字段查询认证结果，拿到真人素材组 GroupId。

参数类型说明
byted_tokenstring，必填从步骤一任务结果中获取的认证令牌。
project_namestring项目名称，默认 default。

步骤三：批量提交真人素材

参数类型说明
group_idstring，必填真人素材组 ID，从步骤二认证结果中获取的 GroupId。
project_namestring项目名称，默认 default。
asset_typestring素材类型，默认 Video。可选 Video（视频，默认，推荐用于真人人像）/ Image（图片）。
assetsarray，必填素材列表，支持一次提交多个素材。元素含 url（公网可访问素材 URL，必填）与 name（素材名称，必填）。

调用示例

curl https://api.huanxing.ai/v1/seedance2/real-avatar \
  -H "Authorization: Bearer sk-你的令牌" \
  -H "Content-Type: application/json" \
  -d '{
    "group_id": "group_xxx",
    "project_name": "default",
    "asset_type": "Video",
    "assets": [
      { "url": "https://example.com/avatar-a.mp4", "name": "avatar-a" },
      { "url": "https://example.com/avatar-b.mp4", "name": "avatar-b" }
    ]
  }'

响应

字段类型说明
codeinteger响应状态码，成功时为 200。
data.idstring本地任务 ID，用于后续查询素材审核状态。
data.objectstring任务对象类型，固定为 seedance.avatar.asset.task。
data.statusstring任务初始状态，提交后为 processing。
data.progressinteger任务进度（0 ~ 100）。
data.modelstring当前使用的模型名称。

查询审核结果

素材提交后为异步审核任务，使用获取任务状态接口查询：GET /v1/tasks/{id}。

全部通过

  • result.usable_assets[].asset_url 可直接用于 Seedance 2.0 视频生成。
  • result.failed_assets 中的素材需更换源文件或重新提交。

审核后使用素材

审核通过后，将 asset://... URL 直接传入 Seedance 2.0 视频生成接口：

{
  "model": "doubao-seedance-2.0-face",
  "prompt": "使用参考人像生成视频",
  "image_with_roles": [
    { "url": "asset://asset_a", "role": "reference_image" }
  ]
}

真人人像素材建议配合 doubao-seedance-2.0-face 或 doubao-seedance-2.0-fast-face 模型使用。服务端识别到 asset:// 前缀后，会直接提交官方生成任务，不会再次触发素材审核。
