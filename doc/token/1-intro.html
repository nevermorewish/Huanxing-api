模型与令牌

模型广场

模型广场（Model Marketplace）是控制台里所有可用模型的清单。任何接入工作的第一件事，都是先在这里确认「目标模型是否存在」「default 分组是否可用」「价格是不是预期内」。

列表里能看到的信息

字段含义
模型名API 调用时填入的 model 字段，例如 claude-sonnet-4-6、gpt-5.4、gemini-3.1-pro。
上游供应商背后接的是 Anthropic、OpenAI、Google、Azure、AWS Bedrock 等中的哪一家。
所在分组该模型当前可由 default 分组令牌访问。
上下文长度模型一次能接收的最大 token 数，例如 200K、1M。
计费倍率相对官方定价的倍数；输入 / 输出 / 缓存可能有不同倍率。
能力标签是否支持流式（stream）、工具调用（tool_use）、视觉（vision）、缓存（cache）等。

挑选模型的常见思路

  • 代码任务：优先选有「工具调用」能力的旗舰对话模型，如 Claude Sonnet 系列与 GPT-5 系列。
  • 长上下文阅读：选明确标注 200K / 1M 上下文长度的模型，避免传完整代码库时被截断。
  • 批量任务 / 后台脚本：选倍率更低的 mini / haiku / flash 系列模型，并设较短的输出上限。
  • 图像生成 / 编辑：到 绘图模型教程 看专用模型说明。

价格判断小窍门

列表上的「输入 / 输出 / 缓存」三列价格不是绝对值，而是「上游官方价 × 倍率」。比如官方 Sonnet 输入 3 USD / 1M tokens，倍率 1.0x 就是 3 美元，倍率 0.5x 就是 1.5 美元。请按控制台实际展示的倍率判断成本。

看不到目标模型？

  1. 切换「分组」筛选条件为 default，确认当前只查看 default 分组模型。
  2. 到 令牌分组 查看 default 分组说明。
  3. 如果目标模型未在 default 分组开放，请联系管理员加入该模型。
  4. 如果账号权限受限，请联系管理员确认 default 分组是否已开放。
