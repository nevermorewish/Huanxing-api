绘图模型

GPT-Image-2

GPT-Image-2 是 OpenAI 风格的图像模型，长项在文字遵循度、版式构图、文字排版与多步编辑。寰星云科把它挂在 OpenAI 兼容接口下，用标准的 /v1/images/generations 端点调用。

调用示例

curl https://api.huanxing.ai/v1/images/generations \
  -H "Authorization: Bearer sk-你的令牌" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-image-2",
    "prompt": "A vintage poster of a sci-fi convention, with bold title \"NEON FUTURE\" at top, retro 80s palette",
    "size": "1024x1024",
    "quality": "high",
    "n": 1
  }'

支持的尺寸与画幅

size用途
1024x1024正方形，社交媒体头图
1024x1792竖版海报 / 手机封面
1792x1024横版头图 / 网页 Banner

遮罩编辑

把一张图叠加 PNG 透明遮罩，告诉模型「这块区域可以重新画」：

curl https://api.huanxing.ai/v1/images/edits \
  -H "Authorization: Bearer sk-你的令牌" \
  -F image=@source.png \
  -F mask=@mask.png \
  -F prompt="把树替换为一棵樱花树" \
  -F model=gpt-image-2 \
  -F size=1024x1024

遮罩中透明部分代表「允许重画」。

写文字 / 字体

GPT-Image-2 对英文短语处理较好，中文较长字符串建议拆短句、明确字体风格描述（例如「黑体加粗」「书法毛笔」「等宽科技感」）。

避免常见误解

  • 它不是 GPT-4 / GPT-5 文本模型，写代码或回答问题请用对话模型。
  • 不支持流式输出，必须等图全部生成完成再返回。
  • 同一份 prompt 多次请求结果会不同，要稳定结果请固定 seed（如果当前渠道支持）。

如果你需要批量出图，建议先用最便宜的尺寸 + quality=standard 做大量草图筛选，确定方向后再用 high 出最终成品。
