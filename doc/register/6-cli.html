快速开始 · 第 4 步

安装 CLI 工具

前一步「环境检查」完成后（Windows 端装好 Node.js 与 Git Bash），按系统切到对应页签，安装 Claude Code CLI。

安装 Claude Code

安装 Claude Code：打开 PowerShell，运行以下命令：

npm install -g @anthropic-ai/claude-code

这个命令会从 npm 官方仓库下载并安装最新版本的 Claude Code。

将 ~/.local/bin 加入 PATH（仅当提示要求时）：

[Environment]::SetEnvironmentVariable('Path', ([Environment]::GetEnvironmentVariable('Path','User') + ";$HOME\.local\bin"), 'User')

验证 Claude Code 安装：安装完成后，输入以下命令检查是否安装成功：

claude --version

如果显示版本号，恭喜你！Claude Code 已经成功安装了。

安装 Claude Code CLI

打开终端：

安装 Claude Code：在终端运行以下命令：

curl -fsSL https://claude.ai/install.sh | bash

可选项：出现提示后运行提供的命令：

echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc && source ~/.bashrc

验证 Claude Code 安装：

claude --version

如果显示版本号，恭喜你！Claude Code 已经成功安装了。

安装 Claude Code

安装 Claude Code：打开终端，运行以下命令：

curl -fsSL https://claude.ai/install.sh | bash

如果遇到权限问题，可以使用 sudo：

sudo curl -fsSL https://claude.ai/install.sh | bash

验证 Claude Code 安装：

claude --version

如果显示版本号，恭喜你！Claude Code 已经成功安装了。

安装 Codex

Codex 是 OpenAI 官方的终端代码助手，通过 npm 全局安装即可。三个系统命令一致，需要先按上方完成 Node.js 与 npm 的安装。

安装命令：

npm install -g @openai/codex@latest

验证安装：

codex --version

如果显示版本号，说明安装成功。

Codex 接入寰星云科的接口与令牌配置在 CLI 配置教程 · Codex 或 CC-Switch · Codex 中详细说明。

安装 Gemini CLI

Gemini CLI 是 Google 官方的终端 AI 助手，同样通过 npm 全局安装。

安装命令：

npm install -g @google/gemini-cli@latest

验证安装：

gemini --version

如果显示版本号，说明安装成功。

Gemini CLI 接入寰星云科的接口与令牌配置在 CLI 配置教程 · Gemini 或 CC-Switch · Gemini 中详细说明。
