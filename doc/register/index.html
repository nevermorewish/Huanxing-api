Getting Started

快速开始

本章把第一次使用寰星云科 API 拆成 4 步。每一步都对应导航中的一个子页面，按顺序完成即可在 Claude Code、Codex 或 Gemini CLI 上跑通第一次调用。

1

购买额度

进入「钱包管理」选择或自定义额度后完成支付。

 2

创建 API 令牌

按用途创建独立令牌，统一选择 default 分组，并设置额度与有效期。

 3

环境检查（必要）

检查 Node.js、npm 以及目标 CLI 是否已经安装到位。

 4

安装 CLI 工具

按 Windows / macOS / Linux 安装 Claude Code 等 CLI 工具。
建议：

第一次接入时，先用一个无限制的临时令牌跑通流程，确认能成功调用后，再为生产环境另建一个有额度上限、default 分组、可随时停用的正式令牌。
