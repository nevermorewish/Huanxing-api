快速开始 · 第 3 步

环境检查

把令牌写入客户端之前，先在本机完成 Claude Code 的安装与环境变量设置。下面按 Windows / macOS / Linux 给出图文步骤，按你的系统切换页签即可。

1. 安装 Node.js 环境

Claude Code 需要 Node.js 环境才能运行。

Node.js 环境安装步骤：

  • 打开浏览器访问 https://nodejs.org/
  • 点击「LTS」版本进行下载（推荐长期支持版本）
  • 下载完成后双击 .msi 文件
  • 按照安装向导完成安装，保持默认设置即可

Windows 注意事项：

  • 建议使用 PowerShell 而不是 CMD
  • 如果遇到权限问题，尝试以管理员身份运行
  • 某些杀毒软件可能会误报，需要添加白名单

验证安装是否成功：打开 PowerShell 或 CMD，输入以下命令：

node --version
npm --version

如果显示版本号，说明安装成功。

2. 安装 Git Bash

Windows 环境下需要使用 Git Bash 安装 Claude Code。安装完成后，环境变量设置和使用 Claude Code 仍然在普通的 PowerShell 或 CMD 中进行。

下载并安装 Git for Windows：

  • 访问 https://git-scm.com/downloads/win
  • 点击「Download for Windows」下载安装包
  • 运行下载的 .exe 安装文件
  • 在安装过程中保持默认设置，直接点击「Next」完成安装

验证 Git Bash 安装：打开 Git Bash，输入以下命令验证：

git --version

如果显示版本号，说明安装成功。

1. 安装 Node.js 环境

Claude Code 需要 Node.js 环境才能运行。macOS 可通过 Homebrew 或官方 .pkg 安装。

方式一：Homebrew

brew install node

方式二：官网下载安装包

  • 打开浏览器访问 https://nodejs.org/
  • 点击「LTS」版本下载 macOS 安装包（.pkg）
  • 双击安装包按向导完成安装，保持默认设置即可

验证安装是否成功：打开终端，输入以下命令：

node --version
npm --version

如果显示版本号，说明安装成功。

2. 安装 Git

大多数 macOS 默认随 Xcode Command Line Tools 自带 Git。先用 git --version 检查，未安装再按下面方式补装。

方式一：Xcode Command Line Tools

xcode-select --install

系统会弹出安装窗口，按提示完成即可，Git 会随之提供。

方式二：Homebrew

brew install git

验证 Git 安装：

git --version

如果显示版本号，说明安装成功。

3. macOS 常见问题解决

macOS 安全设置阻止运行：如果系统阻止运行 Claude Code：

  • 打开「系统偏好设置」→「安全性与隐私」
  • 点击「仍要打开」或「允许」
  • 或者在 Terminal 中运行：sudo spctl --master-disable

1. 安装 Node.js 环境

Claude Code 需要 Node.js 环境才能运行。不同发行版的常用安装方式：

Ubuntu / Debian（NodeSource 仓库）：

curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt install -y nodejs

CentOS / RHEL / Fedora：

curl -fsSL https://rpm.nodesource.com/setup_lts.x | sudo bash -
sudo dnf install -y nodejs

通用：nvm（推荐，不需要 sudo 写系统目录）：

curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
source ~/.bashrc
nvm install --lts

验证安装是否成功：

node --version
npm --version

如果显示版本号，说明安装成功。

2. 安装 Git

大多数发行版默认已安装 Git。先用 git --version 检查，未安装再按下面方式补装：

Ubuntu / Debian：

sudo apt update
sudo apt install -y git

CentOS / RHEL / Fedora：

sudo dnf install -y git

Arch Linux：

sudo pacman -S git

验证 Git 安装：

git --version

如果显示版本号，说明安装成功。

3. Linux 常见问题解决

缺少依赖库：某些 Linux 发行版需要安装额外依赖：

# Ubuntu/Debian
sudo apt install build-essential

# CentOS/RHEL
sudo dnf groupinstall "Development Tools"
