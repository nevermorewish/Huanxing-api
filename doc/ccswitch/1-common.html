CC-Switch介绍

Claude Code / Codex / Gemini CLI 全方位辅助工具





下载地址：https://github.com/farion1231/cc-switch/releases/latest

从供应商切换器到 AI CLI 一体化管理平台

统一管理 Claude Code、Codex 与 Gemini CLI 的供应商配置、MCP 服务器、Skills 扩展和系统提示词。

使用 CC-Switch，您可以：

  • ✅ 一键切换 API 配置 - 在多个 API 提供商之间快速切换
  • ✅ 可视化配置管理 - 通过图形界面轻松管理所有配置
  • ✅ MCP 服务器管理 - 管理 Model Context Protocol 服务器
  • ✅ 系统托盘快捷操作 - 通过托盘菜单快速切换

软件下载

Windows

  1. 打开 https://github.com/farion1231/cc-switch/releases/latest，进入 CC-Switch 的 Github Release 页面

  2. 鼠标滚动到最下方选择适合自己版本的安装包，windows 系统推荐下载普通 msi 后缀的安装包进行安装

  1. 安装后运行 CC-Switch 主程序，界面如下。
MacOS

  • MacOS 安装推荐使用 HomeBrew

  • 开启终端后，分别运行以下命令：

# 添加 tap 源
brew tap farion1231/ccswitch

# 安装 CC-Switch
brew install --cask cc-switch
  • 安装完成后，在“启动台”或“应用程序”文件夹中找到 CC-Switch 并启动。
Linux

重要

以下命令中的文件名包含占位符版本号 x.x.x，请访问 https://github.com/farion1231/cc-switch/releases/latest 页面查看最新版本，并替换为实际的版本号和完整文件名。

Debian/Ubuntu 系统：

# 下载 .deb 包
wget https://github.com/farion1231/cc-switch/releases/latest/download/cc-switch_x.x.x_amd64.deb

# 安装
sudo dpkg -i cc-switch_x.x.x_amd64.deb
