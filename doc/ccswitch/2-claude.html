Claude Code 配置

  1. 打开你下载的 CC Switch 软件，你会看到如下图的初始界面
  1. 在顶部工具类型中，选择“Claude”
  1. 在供应商列表中，选择“自定义配置”
  1. 回顾 创建 API 令牌，在寰星云科中创建 default 分组令牌，点击复制按钮，复制 ApiKey 到剪切板
  1. 下拉模态框，找到“API 请求地址”配置项，填写 https://api.huanxing.ai
  1. 继续找到“API Key”配置项，填入你刚才复制的 ApiKey，再点击右下角“添加”按钮
  1. 添加成功后，在主界面会看到我们配置的供应商，在右侧点击“启用”按钮，显示“使用中”，则配置完成
  1. 点击左上角“设置”按钮，在通用页面下拉找到 跳过 Claude Code初次安装确认 ，务必勾选
  1. 在终端运行 claude，看到对话界面并能正常回复即表示配置完成

使用提醒

配置是否生效，请直接以 Claude Code 内的实际对话结果为准，并在 Claude Code 中完成最终测试。
